# MMBC Conference 2026 — Event Hub

A mobile-first digital event hub for MMBC Conference 2026, accessible via QR code on attendee badges. Built with Next.js and deployed on Vercel.

## Features

- **Home** — Live countdown to the event and quick-access navigation
- **Schedule** — Full session schedule with time and track details
- **Speakers** — Searchable speaker lineup organized by session
- **Sponsors** — Sponsor showcase
- **Floor Plan** — Venue map for easy navigation

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Lucide React](https://lucide.dev/) — icons
